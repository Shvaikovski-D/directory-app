### Обзор страницы
Страница представляет собой административный интерфейс **«Справочник погрузчиков»** с трёхколоночной визуальной структурой: **левое навигационное меню**, **центральная рабочая область** с таблицей погрузчиков и фильтрами, **правый сайдбар** с деталями и журналом простоев выбранного погрузчика. Вверху справа — кнопка **Профиль** для управления учётной записью.

---

### Левая панель навигации
- **Вертикальное меню** фиксированной ширины, расположено слева.
- Список пунктов меню в одну колонку: **Пользователи**, **Уведомления и напоминания**, **Настройки АИС ОГПА**, **Справочник погрузчиков** (выделен), **Резервное копирование и восстановление**, **Справочники**.
- Активный пункт подсвечен (фон/индикатор) и служит для навигации между разделами.
- Предполагаемые элементы поведения: иконки слева от текста, hover-эффект, возможность свернуть панель.

---

### Верхняя панель и панель фильтров
- **Заголовок страницы**: крупный текст **Справочник погрузчиков**.
- Под заголовком — **поле фильтра**: метка **Номер погрузчика** и пустое текстовое поле для ввода.
- Кнопки управления фильтрами в одну строку:
  - **🔍 Искать** — применить фильтр.
  - **❌ Сбросить фильтр** — очистить поле.
  - **Добавить** — создать новую запись погрузчика.
- В правой части верхней панели — **Профиль** (кнопка/меню пользователя).
- Поведение: поиск по номеру, валидация ввода, быстрый доступ к созданию записи.

---

### Центральная таблица погрузчиков
**Структура таблицы** (колонки и назначение):

| **Колонка** | **Назначение** |
|-------------|----------------|
| **Код записи** | Уникальный идентификатор записи |
| **Марка** | Производитель или модель (например, Амкодор) |
| **Номер** | Регистрационный/заводской номер (например, 4565 PH-1) |
| **Грузоподъемность** | Значение в тоннах или кг (например, 2.5) |
| **Активен** | Булево состояние; чекбокс или иконка ✓ |
| **Время и дата** | Время последнего изменения или создания записи |
| **Пользователь** | Кто создал/изменил запись (ФИО) |
| **Действия** | Кнопки для операций: редактировать ✏️, удалить ❌ |

- **Строка данных**: каждая строка — один погрузчик; кликабельная для выбора.
- **Сортировка и фильтрация**: заголовки колонок предполагают сортировку по клику.
- **Пагинация**: при большом объёме данных — постраничная навигация внизу (не показана, но ожидаема).
- **Массовые действия**: чекбоксы для выбора нескольких строк и панель действий (возможна реализация).

### Правый сайдбар Простой по погрузчику
- **Заголовок**: «Простой по погрузчику 4565 PH-1» — контекстно-зависимый, показывает выбранный погрузчик.
- Кнопка **Добавить** для создания записи простоя.
- **Таблица простоев** с колонками:

| **Колонка** | **Назначение** |
|-------------|----------------|
| **Код** | Идентификатор записи простоя |
| **Начало** | Дата и время начала простоя |
| **Окончание** | Дата и время окончания простоя |
| **Время простоя** | Продолжительность (например, 1ч) |
| **Причина** | Краткое описание (например, сломался) |
| **Действия** | Редактировать ✏️, удалить ❌ |

- Поведение: при выборе строки в центральной таблице правый сайдбар обновляется; возможна inline-редакция и подтверждение удаления.
- Отображение суммарных метрик: общее время простоев за период, количество инцидентов (может быть добавлено).
